# Classfolio

A CRUD application for posting, updating, viewing, and organizing student work.

## Technologies Used
--- 
Express.js, node.js, Bootstrap, ejs, bcrypt, cloudinary, Javascript, HTML, CSS, Mongodb and mongoose
